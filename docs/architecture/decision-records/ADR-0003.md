# ADR-0003: WGS84 as Canonical Geodetic Datum

**Status:** Accepted  
**Date:** 2026-04-20

## Context
Geographic types must adopt a single datum to avoid silent inconsistencies between packages.

## Decision
All coordinates use WGS84 (lat ∈ [-90, 90], lon ∈ [-180, 180]). `EarthConstants` exposes WGS84 parameters. Haversine distances use `MeanRadiusMeters`.

## Consequences
- Compatible with GPS receivers, Google Maps, HERE Platform APIs.
- Magnetic declination, UTM/MGRS conversions deferred to future packages.
