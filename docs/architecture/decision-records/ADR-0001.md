# ADR-0001: Multi-Target netstandard2.0, netstandard2.1, net8.0

**Status:** Accepted  
**Date:** 2026-04-20

## Context
The HERE.Sdk.Premium ecosystem must support Xamarin.Forms 5 (netstandard2.0), .NET MAUI 8 (net8.0), and the intermediate netstandard2.1 bridge.

## Decision
Target `netstandard2.0;netstandard2.1;net8.0`. Use polyfills (`IsExternalInit`) for newer C# features on older TFMs.

## Consequences
- Broadest consumer compatibility at the cost of minor polyfill overhead.
- `IsTrimmable` and AOT analysis are gated to net8.0 only.
