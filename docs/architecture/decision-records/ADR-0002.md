# ADR-0002: Zero Runtime Dependencies

**Status:** Accepted  
**Date:** 2026-04-20

## Context
Consumers must not inherit transitive NuGet dependencies from Common.

## Decision
No `PackageReference` items without `PrivateAssets="all"` in the main project. `build/clean-architecture-check.sh` enforces this on every CI run.

## Consequences
- Smaller binary size (target ≤ 100 KB nupkg).
- All geographic calculations implemented in pure .NET (Haversine, FlexiblePolyline).
