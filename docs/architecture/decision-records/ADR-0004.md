# ADR-0004: Immutable Value Objects via `record` / `record struct`

**Status:** Accepted  
**Date:** 2026-04-20

## Context
Geographic primitives and unit types must be safe to share across threads and to use as dictionary keys.

## Decision
All domain primitives are `readonly record struct` or `sealed record`. Invariants are enforced in the primary constructor. No mutable setters.

## Consequences
- Thread-safe by construction.
- Value equality and `GetHashCode` generated by the compiler.
- Requires `IsExternalInit` polyfill on netstandard2.0/2.1.
