## Why

Common's governance spec was behind the sibling repos' specs, and several
enforcement mechanisms were missing or non-functional:

1. `.githooks/` directory was absent despite `core.hooksPath = .githooks` being set.
   Hooks existed only in `.git/hooks/` (auto-generated, non-versioned, not distributed).
2. Pre-push stage was wired but empty — no hooks ran at push time.
3. `.pre-commit-config.yaml` was missing four hooks specified in `ecosystem-overview.md`:
   `cspell`, `dotnet format`, `markdownlint`, `yamllint`.
4. No package restore check in locked mode (phantom dependencies undetected locally).
5. No XML documentation check locally (only in CI).
6. `CLAUDE.md` referenced `main` instead of `master` for branch protection.

## What changes

### ADDED `.githooks/` (3 files)

Canonical bash scripts from `Here.Sdk.Meta/scripts/governance/hooks/`:

- `pre-commit`: branch-guard (blocks commits on master/develop/release/*) +
  worktree-convention warning + delegates to `pre-commit run`.
- `pre-push`: branch-guard (blocks pushes to protected branches) +
  delegates to `pre-commit run --hook-stage pre-push` + native fallbacks
  (package restore, XML doc check).
- `commit-msg`: delegates to `pre-commit run --hook-type commit-msg`.

All three work without `pre-commit` installed (native bash fallback).

### MODIFIED `.pre-commit-config.yaml`

Added hooks aligned with canonical Meta config:

- `cspell` (local node hook, requires `.cspell.json`)
- `dotnet-format` (local system hook, C# files only)
- `markdownlint` (via `igorshubovych/markdownlint-cli` v0.43.0)
- `yamllint` (via `adrienverge/yamllint` v1.35.1)
- `dotnet-restore-locked` (pre-push stage, locked-mode restore)
- `dotnet-doc-check` (pre-push stage, CS1591 error check)
- `coverage-gate` (pre-push stage, runs `./build/coverage-gate.sh`)

### MODIFIED `Directory.Build.props`

Added `<RestorePackagesWithLockFile>true</RestorePackagesWithLockFile>` to enable
locked-mode restore for phantom-dependency detection.

### ADDED `packages.lock.json` (3 files)

Generated by `dotnet restore` after enabling `RestorePackagesWithLockFile`:

- `src/Here.Sdk.Common/packages.lock.json`
- `tests/Here.Sdk.Common.UnitTests/packages.lock.json`
- `tests/Here.Sdk.Common.Benchmarks/packages.lock.json`

### MODIFIED `CLAUDE.md`

Line 8: `main` corrected to `master` in branch protection rule.

### MODIFIED `openspec/specs/governance/spec.md`

Added six new requirements (see delta spec in this proposal's `specs/` folder).
